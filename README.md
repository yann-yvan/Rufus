## Usage

### Step 1:
#### Add it in your root build.gradle at the end of repositories
```
allprojects {
  repositories {
	...
	maven { url 'https://jitpack.io' }
  }
}
```

### Step 2:
#### Add gradle dependecy
```
dependencies {
   implementation 'com.github.yann-yvan:Rufus:V1.0'
}
```
